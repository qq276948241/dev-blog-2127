## 1. 产品概述

面向程序员的个人技术博客网站，用于记录和分享技术笔记与学习心得。采用深色主题打造极客风格的阅读体验，支持文章分类、标签筛选和代码高亮。

- 核心目的：为开发者提供个人技术沉淀和知识分享的平台
- 目标用户：程序员、技术爱好者、技术学习者

## 2. 核心功能

### 2.1 功能模块

1. **首页**：Hero 区域、文章列表、分类筛选、标签筛选、搜索
2. **文章详情页**：Markdown 渲染、代码高亮、目录导航、相关文章推荐
3. **分类页面**：所有分类列表、分类下的文章集合
4. **标签页面**：所有标签云、标签下的文章集合
5. **归档页面**：按时间线展示所有文章
6. **关于我页面**：个人简介、技能栈、工作经历、项目经历、联系方式（简历风格）
7. **搜索页面**：全站文章搜索结果

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深黑灰（#0a0a0f）作为背景，深蓝紫（#6366f1）作为主强调色，青绿（#22d3ee）作为次强调色
- **辅助色**：琥珀色（#f59e0b）用于标签，玫瑰色（#f43f5e）用于热门标识
- **按钮风格**：圆角胶囊按钮，hover 时有发光效果
- **字体**：标题使用 Space Grotesk，正文使用 JetBrains Mono，代码使用 Fira Code
- **布局风格**：双栏布局（主内容 + 侧边栏），卡片悬浮玻璃态效果
- **视觉元素**：渐变光晕背景、噪点纹理、网格背景、光标跟随效果
- **图标风格**：Lucide 线性图标

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|-----------|-------------|-------------|
| 首页 | Hero 区域 | 渐变光晕背景、打字机动画、头像脉冲效果、社交链接悬浮放大 |
| 首页 | 文章卡片 | 玻璃态卡片、hover 上移发光、封面渐变遮罩、分类标签胶囊 |
| 首页 | 筛选栏 | 磨砂玻璃导航条、标签 chip 选中态发光 |
| 首页 | 侧边栏 | 固定悬浮、热门文章排名序号高亮、标签云多彩渐变 |
| 文章详情页 | 代码块 | 深色 IDE 风格、行号显示、语言标签、复制按钮、语法高亮 |
| 文章详情页 | 目录导航 | 滚动跟随、当前章节左侧色条高亮 |
| 分类页面 | 分类卡片 | 图标 + 名称 + 数量、渐变边框、hover 发光 |
| 标签页面 | 标签云 | 不同字号、多彩渐变标签、随机旋转角度 |
| 归档页面 | 时间线 | 纵向时间轴、年份节点发光、月份折叠展开 |
| 关于我页面 | 技能条 | 渐变进度条、发光动画 |
| 关于我页面 | 工作经历 | 纵向时间线、公司 logo 占位、项目标签 |

### 4.3 响应式设计

- 桌面端（1024px+）：双栏布局，侧边栏固定，主内容区最大宽度 1200px
- 平板端（768px-1024px）：侧边栏折叠为顶部抽屉，单栏布局
- 移动端（<768px）：单栏流式布局，汉堡菜单导航，卡片间距缩小
- 触摸优化：增大点击区域至 44x44px，卡片 hover 改为 active 态

### 4.4 动效设计

- 页面进入：渐入 + 轻微上滑动画，内容交错延迟
- 卡片悬浮：上移 4px + 模糊发光阴影
- 滚动动画：侧边栏目录高亮跟随、视差背景
- 打字机效果：Hero 区域签名逐字打印
- 代码块：复制成功提示、行号悬停效果
